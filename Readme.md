# Case II : Customer Membership

* Repository ini berisi file python yang membuat simple program 
untuk masalah Case II, yaitu Customer Membership di pembelajaran
Python Software Engineering di platform PacMann.

* Beberapa _objective_ dari program tersebut adalah
  1. Membuat fitur di mana _user_ bisa melakukan **show all tier and membership** 
     yang tersedia di PacCommerce.
  2. _user_ bisa melakukan **show ketentuan jumlah** monthly expense dan monthly income
    yang dibutuhkan dari setiap membership.
  3. _predict user_ masuk ke dalam membership mana berdasarkan parameter expense dan income.
  4. _calculate_ total belanja akhir berdasarkan tier membership PacCommerce.

* Program menggunakan OOP dan dalam hal ini yang menjadi class adalah
  User dengan nama class _MembershipUser_.

* class MembershipUser:
  Atribut-atributnya adalah username, monthly_expense, monthly_income.
  Methode-methodnya adalah show_benefit, show_requirements, predict_membership,
  calculate_price.
